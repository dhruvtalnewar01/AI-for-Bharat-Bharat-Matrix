# Bharat MatrixAI

## Bharat MatrixAI — The First Agentic Interface for India

A proprietary Cognitive Visual Engine navigating Indian digital infrastructure autonomously.

### Architecture

- **frontend-web/** — Next.js 16 cinematic web experience (GSAP, Three.js, Framer Motion)
- **frontend-app/** — React Native / Expo mobile application
- **backend-agent/** — FastAPI backend with browser automation, privacy guardrails, and vernacular pipeline

### Getting Started

```bash
# Frontend Web
cd frontend-web
npm install
npm run dev
```

### Deployment (Netlify)

```bash
cd frontend-web
npm run build   # Produces static `out/` folder
# Deploy the `out/` folder to Netlify
```

### Environment Variables

Create `frontend-web/.env.local`:
```
NEXT_PUBLIC_VAPI_KEY=<your-vapi-key>
NEXT_PUBLIC_VAPI_ASSISTANT_ID=<your-assistant-id>
```
